# 基金估计计算App - 需求分析文档

## 一、项目背景

### 1.1 项目概述
基金估计计算App是一款面向**个人投资者**的基金收益计算工具，旨在帮助用户快速、准确地估算基金收益、管理持仓组合。

### 1.2 目标用户
- **个人投资者**：界面简洁易用，功能实用直观
- 需要快速估算基金收益的用户
- 需要管理多个基金持仓的用户

---

## 二、技术选型

### 2.1 前端技术栈
| 技术 | 选择 | 说明 |
|------|------|------|
| 框架 | Vue 3 | 渐进式框架，Composition API |
| 构建工具 | Vite | 快速开发构建 |

| 数据存储 | localStorage / IndexedDB | 本地持久化 |

### 2.2 设备适配
- **响应式设计**：同时适配PC端和移动端

## 四、核心功能模块

### 4.1 基金查询模块
**功能描述**：通过基金代码或名称查询基金信息

| 子功能 | 说明 |
|--------|------|
| 基金搜索 | 输入基金代码/名称搜索 |
| 基金详情 | 显示基金类型、净值、涨跌幅等 |
| 历史净值 | 查看历史净值走势 |
| 基金对比 | 多只基金对比分析 |

### 4.2 收益计算模块
**功能描述**：计算基金投资收益

| 子功能 | 计算公式 | 说明 |
|--------|----------|------|
| 单次收益 | (当前净值-买入净值) * 份额 | 一次性买入收益 |
| 定投收益 | 复杂计算 | 定期定额投资收益 |
| 持有收益率 | 收益/本金 * 100% | 百分比展示 |
| 年化收益率 | 根据持有时间换算 | 年化收益展示 |

### 4.3 持仓管理模块
**功能描述**：管理个人基金持仓组合

| 子功能 | 说明 |
|--------|------|
| 添加持仓 | 录入买入信息（代码、金额、日期、份额） |
| 持仓列表 | 展示所有持仓基金 |
| 持仓编辑 | 修改/删除持仓记录 |
| 加仓/减仓 | 记录后续操作 |
| 持仓分布 | 按类型展示持仓占比（饼图） |
| 总资产统计 | 汇总总投入、总收益、总市值 |

### 4.4 数据可视化模块
**功能描述**：图表展示基金数据

| 图表类型 | 用途 |
|----------|------|
| 折线图 | 净值走势、收益趋势 |
| 饼图 | 持仓分布、资产配置 |
| 柱状图 | 收益对比、涨跌幅对比 |
| 数据表格 | 详细数据展示 |

### 4.5 自选模块 ✅
**功能描述**：收藏关注的基金，快速查看涨跌情况

| 子功能 | 说明 |
|--------|------|
| 添加自选 | 从搜索页添加基金到自选列表 |
| 自选列表 | 展示所有自选基金及实时涨跌 |
| 持有标签 | 标识已持仓的自选基金 |
| 分类筛选 | 全部/最近/持有 三种筛选 |
| 删除自选 | 从自选列表移除基金 |

## 五、数据结构设计

### 5.1 基金信息
```typescript
interface Fund {
  code: string;           // 基金代码
  name: string;           // 基金名称
  type: string;           // 基金类型（股票型/债券型/混合型等）
  currentNetValue: number; // 当前净值
  dailyChange: number;    // 日涨跌幅
  netValueDate: string;   // 净值日期
}
```

### 5.2 持仓记录
```typescript
interface Position {
  id: string;             // 记录ID
  fundCode: string;       // 基金代码
  buyDate: string;        // 买入日期
  buyNetValue: number;    // 买入净值
  shares: number;         // 份额
  amount: number;         // 买入金额
  remark?: string;        // 备注
}
```

### 5.3 自选基金
```typescript
interface WatchlistItem {
  id: string;             // 记录ID
  fundCode: string;       // 基金代码
  fundName: string;       // 基金名称
  fundType: string;       // 基金类型（关联板块）
  addedAt: string;        // 添加时间
  groupId?: string;       // 所属分组ID
}

interface WatchlistGroup {
  id: string;             // 分组ID
  name: string;           // 分组名称
  order: number;          // 排序
}
```

### 5.4 本地存储结构
```typescript
interface LocalStorage {
  positions: Position[];  // 持仓列表
  fundCache: Record<string, Fund>; // 基金信息缓存
  settings: UserSettings; // 用户设置
  watchlist: WatchlistItem[]; // 自选列表
  watchlistGroups: WatchlistGroup[]; // 自选分组
}
```
